- variables is building block of any programming language.
- it is used to store value .basically store value in memory.
- it is like box where we store things labelled by their name so we can easily find by name whenever it needed.

# Rules to naming variables

- cannot be reserved keyword - if, for, while, new
- should be meaningful - not like x, a, b
- cannot start with number not like(1name)
- cannot contain space or hyphen
- case sensitive
