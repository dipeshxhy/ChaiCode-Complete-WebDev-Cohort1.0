Category format extension description
Audio MP3 .mp3 Most common audio format
WAV(Waveform Audio File Format) .wav Uncompressed audio format
lossless format standard for Windows. It retains the original audio quality without any data loss, resulting in very large file sizes. Used primarily for professional audio editing and archiving.
AAC(Advanced Audio Coding) .aac / .m4a Advanced audio coding format
A newer, lossy compression format that generally achieves better sound quality than MP3 at similar or lower bitrates. It is the standard format for iTunes, Apple Music, and YouTube audio.
